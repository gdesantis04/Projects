# Fall 2025 Engineering Projects

This repository contains selected engineering projects from Fall 2025, organized into separate project folders.

## Projects

### 1. Lowpass Filter Design
Design of a 50-ohm microstrip low-pass filter on FR-4 with a 3 GHz cutoff frequency, Chebyshev response, and attenuation verification.

### 2. CMS ETL Test Stand Design Review
Capstone design review for an automated test stand supporting verification of RB3, RB6, and RB7 readout boards for the CMS Endcap Timing Layer upgrade.

### 3. ASIC Emulator Final Project
ELEC 434 final project involving an ASIC emulator for voltage monitoring, 500 kHz clock generation, and synchronous power-good output.
